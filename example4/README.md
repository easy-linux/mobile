# Example 3 for Cordova

## Как пользоваться

    npm install
    cd cordova
    cordova prepare
    cd ..
    npm run android-emulate
    npm run ios-emulate

## Команды для сборки дев и прод версий

    npm run ios_dev
    npm run ios_prod

    npm run android_dev
    npm run android_prod
